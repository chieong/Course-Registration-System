Project Description:

This is a Course Registration System supporting students, instructors, and administrators. It includes course registration, timetable management, plan handling, and administrative functions with a command line or web interface

----------------------------------------

How to compile & package src into jar:

Windows: ./mvnw.cmd clean package
Linux: ./mvnw clean package

----------------------------------------
How to Run:

1. Open a command line interpreter in the Release directory
2. Run:
  java -jar .\course-registration-system-0.0.1-SNAPSHOT.jar
 
Requirements:
- Java 17 or above
