How to Run (CLI Version):

1. Go to Release folder
2. Run:
  java -jar .\course-registration-system-0.0.1-SNAPSHOT.jar --cli
 
Requirements:
- Java 17 or above
- No external setup required
- SQLite database included

----------------------------------------

How to Run (Web-base Version):

1. Go to Release folder
2. Run:
   ./mvnw clean package
 	
Requirements:
- Java 17 or above
- No external setup required
- SQLite database included
